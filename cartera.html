Instrucciones
GUÍA DE USO · FÓRMULAS · ESCALABILIDAD
01
Clientes
Ingresá cada cliente con ID único (ej: CL001). El Score se calcula automáticamente según % de cuotas atrasadas: Verde 80–100 / Amarillo 50–79 / Rojo <50.
02
Préstamos
Completá ID, Cliente, Fecha, Monto, Tasa mensual, Plazo, Tipo (Simple/Compuesto). El Monto Total y ROI se calculan automáticamente. Actualizá el Estado a Mora o Finalizado según corresponda.
03
Plan de Pagos
Cronograma de cuotas con amortización. Estado por cuota: Pendiente / Pagado / Atrasado. Actualizá el estado al registrar cada pago en la hoja correspondiente.
04
Pagos Realizados
Registrá cada cobro: ID cuota, fecha, monto, método. Los días de atraso y diferencia vs cuota se calculan solos. Semáforo: Al día / Leve ≤7d / Moderado ≤30d / Crítico +30d.
05
Dashboard
Todos los KPIs (ROI, mora, capital, ganancia) se actualizan en tiempo real al agregar o modificar datos. El resumen de préstamos muestra el estado completo de la cartera.
06
Escalabilidad
El sistema soporta +1000 préstamos. Todos los datos se guardan en LocalStorage del navegador. Para compartir entre dispositivos, exportar como JSON o conectar a Google Sheets.
